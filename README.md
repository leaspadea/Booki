# 🏨 Booki

> Site responsive d'un comparateur d'hébergements et d'activités touristiques. Projet d'intégration web réalisé dans le cadre de la formation Intégrateur Web chez OpenClassrooms.

## 🚀 Démo en ligne

👉 [Voir le site](https://leaspadea.github.io/Booki/)

## 🎯 Contexte du projet

Booki est une agence souhaitant développer un site web permettant aux utilisateurs de trouver des hébergements et activités dans la ville de leur choix.

**Mission :** intégrer une maquette Figma en site responsive multi-supports (mobile, tablette, desktop) à partir d'une base HTML/CSS desktop fournie par "le client".

## 🛠️ Technologies utilisées

- **HTML5** — structure sémantique (header, main, sections, articles, footer)
- **CSS3** — Flexbox, Grid, media queries
- **Font Awesome** — bibliothèque d'icônes
- **Google Fonts** — typographie (Raleway)
- **Git / GitHub** — versioning et déploiement

## ✨ Fonctionnalités principales

- ✅ Site responsive multi-supports (mobile, tablette, desktop)
- ✅ Navigation principale et menu de filtres
- ✅ Affichage dynamique de cartes d'hébergements et d'activités
- ✅ Animations CSS au survol (hover) sur les éléments interactifs
- ✅ Code conforme aux standards W3C
